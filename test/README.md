# Testes do VSCode

## Conteúdo

Esta pasta contém os diversos runners de teste do VSCode. Consulte a documentação em cada subpasta para saber como executá-los:

- `unit`: suíte de testes unitários ([README](unit/README.md))
- `integration`: suíte de testes de API ([README](integration/browser/README.md))
- `smoke`: suíte de testes automatizados de UI ([README](smoke/README.md))
